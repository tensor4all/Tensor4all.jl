SimpleTT

Tensor4all.SimpleTT.SimpleTensorTrain — Type
SimpleTensorTrain{T<:Union{Float64, ComplexF64}}

A simple tensor train (TT/MPS) with statically determined shapes.

SimpleTensorTrain is a simple tensor train library where site dimensions are fixed at construction time.

Supports Float64 and ComplexF64 values.
